# JP Portfolio Website Media

Dieser Ordner ist als Upload-Vorlage fuer das GitHub-Repository `jp-portfolio-website-media` gedacht.

## Struktur

- `images/arbeiten/kategorie-01` bis `images/arbeiten/kategorie-06`: je sechs austauschbare Arbeiten als `arbeit-01.jpg` bis `arbeit-06.jpg`
- `images/hintergruende`: Hintergrundbilder der Haupt-, Info- und Ueber-Mich-Ansicht
- `images/portraits`: Profilbilder
- `images/favicon`: Favicons
- `videos/arbeiten`: Platz fuer spaetere Arbeitsvideos
- `videos/hintergruende`: Platz fuer spaetere Hintergrundvideos
- `texts/site-texts.json`: zentrale Textdatei fuer Titel, Navigation, Beschreibungen, Buttons und Info-Texte

## Austausch

Damit die Webseite ohne Codeaenderung weiter funktioniert, sollen Dateien moeglichst unter gleichem Dateinamen ersetzt werden. Beispiel: Fuer die zweite Arbeit in Kategorie 04 die neue Datei als `images/arbeiten/kategorie-04/arbeit-02.jpg` hochladen.

Wenn neue Texte, Buttons oder Ansichten dazukommen, gehoeren sie in `texts/site-texts.json`.
