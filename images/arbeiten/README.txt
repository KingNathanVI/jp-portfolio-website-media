BILDER AUSTAUSCHEN

Jede Kategorie besitzt einen eigenen Ordner mit sechs Bild-Slots:
arbeit-01.jpg bis arbeit-06.jpg

Zum Austauschen ein neues Bild in den gewünschten Kategorieordner legen und
auf den Namen des zu ersetzenden Slots umbenennen. Am Code ist keine Änderung
notwendig. Für eine saubere Darstellung sollten alle Bilder dasselbe
Seitenverhältnis verwenden.
